解决Vbox屏幕分辨率问题

　　在使用Vbox的过程中发现屏幕分辨率总是非常小，这对于在虚拟机中调程序非常的不便，所以网上查找了增大屏幕分辨率的方法，这里做一下记录，方便以后操作。

　　要增大屏幕分辨率，我们需要安装增强功能，打开虚拟机，点击“设备”->“安装增强功能”(找不到时，按HostKey+C可以显示虚拟机上方的菜单)，桌面会出现VBOXADDITIONS光盘文件。

　　在安装之前，我们需要安装一些必要的库文件：

yum update
yum install gcc
yum install kernel-devel
reboot生效。

　　另外，还要做一个链接，不然安装的时候会出现错误：

ln -s /usr/src/kernels/2.6.32-504.12.2.el6.i686 /usr/src/linux

　　其中2.6.32-504.12.2.el6.i686为内核版本号，根据实际情况填入。
　　到这里，已经可以正常安装增强功能了。

　　打开VBOXADDITIONS光盘文件，点击运行autorun.sh文件就能自动安装。
　　最后重启虚拟机就能看到变化。VBOXADDITIONS不仅能增强虚拟机的分辨率，还可以实现虚拟机和主机之间的自由切换，再也不要靠Host键来切换，主机与虚拟机之间文本的复制也可以直接进行，非常方便。
